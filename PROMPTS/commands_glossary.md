# Commands Glossary

Players may take any of the following actions (combine as needed):

- Move
- Investigate
- Attack
- Search
- Parley
- Trade
- Hide
- Track
- Study
- Use item
- Cast spell
- Social actions (befriend, intimidate, inspire, deceive)
- World interactions (navigate hazards, tend wounds, prepare camp, set watches)

The DM must end each scene with a clear actionable branch prompt referencing these options.
